# **App Name**: TechFlow

## Core Features:

- Service Request Submission: Customers can submit service requests with detailed descriptions of their hardware or software issues, including device type, OS version, and error messages.
- Admin Dashboard: An admin dashboard to view and manage incoming service requests.  The dashboard includes request filtering, technician assignment, and progress tracking.
- Real-time Progress Updates: Customers receive real-time updates on the progress of their repair or service request, including status changes, technician notes, and estimated completion time.
- Booking Appointments: Customers can book appointments for on-site service or drop-off based on technician availability.
- Technician Assignment AI Tool: An AI tool suggests technician assignments based on expertise and workload.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) for a modern tech vibe.
- Background color: Light gray (#E0E0E0) for a clean, professional look.
- Accent color: Soft Green (#8FF7A7) for CTAs and highlights.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective look that supports easy readability on various screens.
- Use hardware-themed icons for service categories and status indicators.
- A clean, grid-based layout for a professional, organized feel.
- Smooth transitions and loading animations to enhance user experience.
